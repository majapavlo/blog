Some of my favourite resources

List of select sources that have been my favourite and most helpful learning material

 

Deep Learning Andrew Trask

Most approachable Deep Learning book out there if you have no engineering or mathematical background. Goes into deeply into details but rather than explaining the maths Trask takes a more intuitive approach to explaining how neural nets are built. Great if you want to code a neural network from start to finish.

 

Maths for Machine Learning

Great accompanied while doing the Imperial Online Course on Coursera. Essential basics if you’re rusty on your maths or just want a simple repetition.

 

Deep Learning for Coders with fastai and PyTorch

Currently going through this one.

 

Machine Learning Introductory Book

 

Deep Learning Bible

Mostly use it as a reference to read up on concepts
